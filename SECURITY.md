# Security Policy

## Supported Versions

Currently, only the main branch of this repository is actively supported with security updates.

## Reporting a Vulnerability

If you discover a security vulnerability within this project, please send an e-mail to x@xscriptor.com or open an issue if it does not contain sensitive payload information.

We take all security vulnerabilities seriously and will aim to respond as quickly as possible. Please do not report security vulnerabilities through public GitHub issues if they pose a significant risk; instead, contact the maintainer directly via email.
