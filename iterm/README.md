# Install the Xscriptor color scheme in iTerm2

- Open iTerm2.
- Go to `Preferences` → `Profiles` → `Colors`.
- Click `Color Presets…` → `Import…`.
- Select the file `iterm/xscriptor.itermcolors` from this project.
- Back to `Color Presets…` and choose the imported preset.
- Optional: set that profile as default under `Profiles`.

References: iTerm2 docs on presets and `.itermcolors` import, and community color scheme installation guide.
