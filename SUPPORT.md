# Support

If you need help with Terminal Xscriptor, you can seek assistance through the following channels:

## Issue Tracker

If you have found a bug or have a feature request, please search the issue tracker to see if someone else has already reported it. If not, feel free to open a new issue.

## Pull Requests

If you have fixed a bug or implemented a new feature, we encourage you to submit a Pull Request. Please make sure to read our `CONTRIBUTING.md` before submitting.

## Community

Discussions, ideas, and general questions should be opened as an issue or in a discussion thread if available on the repository.
