# MobaXterm Color Themes (Xscriptor)

This folder contains ready-to-import color themes for MobaXterm using the native `.mxtcolors` format, plus guidance for manual installation.

## Import (.mxtcolors)
- Open MobaXterm
- Go to Settings → Configuration → Terminal
- Click “Default terminal color settings” → “Import a color theme from a file”
- Select any theme from
- Apply and restart MobaXterm if needed
- Ensure saved sessions use the “Default” color scheme or customize per-session under Session settings → Terminal → Colors
