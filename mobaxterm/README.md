# MobaXterm Color Themes (Xscriptor)

This folder contains ready-to-import color themes for MobaXterm using the native `.mxtcolors` format, plus guidance for manual installation.

## Files
- Location: `./themes`
- Theme files:
  - `x.mxtcolors`
  - `madrid.mxtcolors`
  - `lahabana.mxtcolors`
  - `seul.mxtcolors`
  - `miami.mxtcolors`
  - `paris.mxtcolors`
  - `tokio.mxtcolors`
  - `oslo.mxtcolors`
  - `helsinki.mxtcolors`
  - `berlin.mxtcolors`
  - `london.mxtcolors`
  - `praha.mxtcolors`
  - `bogota.mxtcolors`

## Import
- Open MobaXterm
- Go to Settings → Configuration → Terminal
- Click “Default terminal color settings” → “Import a color theme from a file”
- Select the desired `.mxtcolors` file from `./themes`
- Apply; restart MobaXterm if needed
- Ensure saved sessions use the “Default” color scheme or set per-session under Session settings → Terminal → Colors
