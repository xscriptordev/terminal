# Terminal.app Themes (Xscriptor)

Quick Install
- Open Finder and go to terminal/terminal.app/themes
- Double-click any .terminal file to add the profile and open it
- To set it as default: Terminal → Settings → Profiles → select the profile → Default
- To use it without changing the default: Shell → New Window → choose the profile

Location
- Themes (.terminal) are in terminal/terminal.app/themes
- You can import multiple and switch between them from Profiles

Notes
- If the profile doesn’t appear after opening the .terminal, quit and reopen Terminal
